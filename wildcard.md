# Widlcard 


|コマンド|内容|
|:------------:|:-----------|
|*|0以上の文字にマッチ|
|?|1文字マッチ|


## [] character class 

|コマンド|内容|
|:------------:|:-----------|
|[]|いずれかを含む|
|!|いずれも含まない|
 
```
// caで始まり、nかtを含む文字（その後文字あってもなくてもok) => can, cat, candy, catch
ca[nt]*

// []が含まれない文字で始まる => bascketball, cricket
[!aeiou]*

//

```


## 範囲指定

|コマンド|内容|
|:------------:|:-----------|
|-|いずれかを含む|


```
// a,b,c,d,e,f,gのいずれかで始まる
[a-g]*

// 3,4,5,6のいずれかで始まる
[3-6]*

//

```

### 定義された範囲指定


|コマンド|内容|
|:------------:|:-----------|
|[[:alpha:]]|アルファベット(大文字小文字問わず)|
|[[:alnum:]]|アルファベット or 数字(0-9)|
|[[:digit:]]|0-9|
|[[:lower:]]|小文字|
|[[:space:]]|space, tab, 改行|
|[[:upper:]]|大文字|


## escape
* wildcardにマッチする検索をしたいときはエスケープ

```
// ?で終わる => done?
*\?

//

```
